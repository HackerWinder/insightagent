## 🎯 项目概述

## 🚀 快速启动指南

1. **配置环境变量**
   ```bash
   cp .env.example .env
   # 编辑.env文件，添加OpenAI API密钥
   ```

2. **启动系统**
   ```bash
   ./start.sh
   ```

3. **访问应用**
   - 前端: http://localhost:3000
   - 后端API: http://localhost:8000
   - API文档: http://localhost:8000/docs

4. **开始使用**
   - 输入产品名称创建分析任务
   - 系统自动收集和分析市场数据
   - 查看生成的洞察报告
